## Notes

1. **Color Contrast**: Ensure `--brand-green` (#7cb342) has sufficient contrast on white backgrounds for accessibility
2. **Logo Usage**: Use full logo on large surfaces (onboarding), monogram on small surfaces (icon, minimized cube)
3. **Consistency**: All user-facing text should say "Offlyn Apply" not just "Offlyn"
4. **Build Process**: After updating public/icons/, run `npm run build` to copy to dist/icons/
5. **Testing**: Test on both light and dark browser themes to ensure logo visibility
