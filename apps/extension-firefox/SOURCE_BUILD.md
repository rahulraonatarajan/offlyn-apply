# Offlyn Apply — Source Code Build Instructions

## Overview

This extension is written in TypeScript and bundled with esbuild.
The `dist/` folder in the submitted ZIP is generated from the source in `src/` and `public/`.

---

## Requirements

| Tool    | Version   | Install                          |
|---------|-----------|----------------------------------|
| Node.js | v22.x LTS | https://nodejs.org/en/download   |
| npm     | v10+      | Included with Node.js            |
| OS      | macOS, Linux, or Windows (WSL recommended on Windows) |

---

## Step-by-step build instructions

### 1. Install Node.js v22

Download and install from https://nodejs.org/en/download (LTS version).

Verify:
```
node --version   # should print v22.x.x
npm --version    # should print 10.x.x or higher
```

### 2. Install dependencies

```
npm install
```

This installs esbuild, TypeScript, and web-ext from `package.json`.

### 3. Build the extension

```
npm run build
```

This runs `node esbuild.config.mjs`, which:
- Compiles all TypeScript source files in `src/` to JavaScript
- Bundles each entry point into a single minified output file in `dist/`
- Copies all static files from `public/` into `dist/`

### 4. Verify the output

The `dist/` folder produced by this build is identical to the submitted ZIP (minus `.map` files and `test-page.html`, which are excluded from the package).

## Build script summary

```
npm install && npm run build
```

Output: `dist/`

## Third-party files included as-is (not built)

The following files are unmodified copies of open-source libraries and are copied directly from `public/` into `dist/` — they are NOT transpiled or generated by the build:

| File | Library | Version | Source |
|------|---------|---------|--------|
| `onboarding/pdf.min.js` | Mozilla pdf.js (pdfjs-dist) | 3.11.174 | https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js |
| `pdf.worker.mjs` | Mozilla pdf.js worker (pdfjs-dist) | 3.11.174 | https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.worker.min.js |
| `dashboard/chart.umd.min.js` | Chart.js | 4.4.1 | https://cdn.jsdelivr.net/npm/chart.js@4.4.1/dist/chart.umd.min.js |

These files cause the `DANGEROUS_EVAL` lint warnings — this is inherent to the pdf.js library and unrelated to extension code.

## Source structure

```
src/               TypeScript source files
  background.ts    → dist/background.js
  content.ts       → dist/content.js
  popup/           → dist/popup/popup.js
  onboarding/      → dist/onboarding/onboarding.js
  dashboard/       → dist/dashboard/dashboard.js
  settings/        → dist/settings/settings.js
  shared/          Shared utilities and types
  ui/              Content script UI components
  theme/           Theming helpers

public/            Static files copied verbatim into dist/
  manifest.json    Extension manifest
  icons/           Extension icons
  home/            Home page HTML/CSS/JS
  popup/           Popup HTML
  onboarding/      Onboarding page HTML + pdf.min.js
  dashboard/       Dashboard HTML + chart.umd.min.js
  settings/        Settings page HTML
  parse-resume/    Resume parser page
  job-detected/    Job detection page
  privacy/         Privacy policy page
  help/            Help page
  pdf.worker.mjs   pdf.js worker (third-party)
```
